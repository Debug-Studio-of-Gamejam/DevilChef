# DevilChef

一个游戏制作人掉进自己制作的游戏中的故事，必须给魔王做饭就才有可能活着出去！
在八个可互动地点中寻找食材，利用火力和时间烹饪出最佳菜品。与多达15个游戏角色对话，发掘出隐藏在地狱背后的黑暗秘密。通过魔王的挑战，但别忘记自己的身份。发现游戏世界的BUG，最终做出你的选择。
以中世纪恶魔和地狱设定为主要元素，在传说故事的基础上，设计出八位个性迥异、奇诡艳丽的恶魔NPC，人外、覆面、哥特、异头等风格应有尽有。草木虫鱼皆有剧情来源，一个魔幻现实交缠的世界徐徐展开。
本作包含近三十张精美立绘和画面，1万字上下的剧情文本，沉浸式配乐以及大量分支选择和最终的三个结局。
